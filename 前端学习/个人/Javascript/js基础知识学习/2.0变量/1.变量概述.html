js变量概述 在 JavaScript 中，变量用于存储数据值或引用值。变量的使用可以使程序更加灵活和可读性更强。 在 JavaScript 中，可以使用 var、let 和 const 关键字来声明变量。 使用 var 声明的变量是全局或函数作用域的，可以被重新赋值和重新声明。例如： var x = 10; x = 20; var x = 30; 使用 let 声明的变量是块级作用域的，只在当前块级作用域内有效，可以被重新赋值但不能被重新声明。例如： let y = 10; y = 20; 使用 const 声明的变量也是块级作用域的，与 let 类似，但是不能被重新赋值和重新声明。例如： const z = 10; 除了使用关键字来声明变量，还可以通过省略关键字直接声明变量，这种方式被称为隐式声明。但是，不推荐使用隐式声明，因为容易造成变量污染和作用域问题。 变量命名需要遵循一定的规则，例如变量名必须以字母、下划线或美元符号开头，不能以数字开头，变量名区分大小写等等。 在 JavaScript 中，变量还有一些常见的类型，例如数字类型、字符串类型、布尔类型、数组类型、对象类型等等。变量的类型可以随时改变，JavaScript 是一种动态类型语言。 -->
